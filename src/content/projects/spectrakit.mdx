SpectraKit is a Python library for vibrational spectroscopy data — IR, Raman, NIR. Every function takes NumPy arrays in, returns NumPy arrays out. Core dependencies are numpy and scipy. Everything else (plotting, CLI, sklearn integration, HDF5) is optional.

## What It Does

- **Baseline correction** — ALS, ArPLS, SNIP, polynomial, rubberband (all with convergence diagnostics)
- **Smoothing** — Savitzky-Golay, Whittaker with wavenumber-aware penalty
- **Normalization** — SNV, min-max, area, L2 vector
- **Derivatives** — Savitzky-Golay, Norris-Williams gap-segment
- **Scatter correction** — MSC, Extended MSC with Legendre basis
- **Transforms** — Kubelka-Munk, ATR correction, absorbance ↔ transmittance
- **Peak analysis** — Detection and integration
- **Similarity** — Cosine, Pearson, spectral angle, Euclidean (batch 2D × 2D)
- **I/O** — JCAMP-DX, SPC, CSV, HDF5, Bruker OPUS (native parser)
- **Pipeline** — Composable chains, sklearn-compatible via `SpectralTransformer`
- **CLI** — `spectrakit info`, `spectrakit convert` for quick file operations
